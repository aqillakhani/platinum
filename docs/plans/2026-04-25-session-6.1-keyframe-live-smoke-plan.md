# Session 6.1 Keyframe Live Smoke Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Replace `RemoteAestheticScorer`'s `NotImplementedError` stub with a real `httpx`-based client backed by an in-repo FastAPI `score_server`, add a `platinum keyframes` CLI, then drive a live 3-scene Cask of Amontillado smoke run on a vast.ai 4090.

**Architecture:** Three layers, each with a Fake/mock pattern proven across S4-S6. (1) `RemoteAestheticScorer` is `httpx.AsyncClient` with `MockTransport` test seam, mirroring `HttpComfyClient`. (2) `scripts/score_server/` ships a FastAPI app + LAION CLIP+MLP via `open_clip` + `torch`, with `Depends`-based scorer injection so tests skip the GPU stack. (3) `KeyframeGeneratorStage` gains a `scene_filter` runtime knob and `scoring_succeeded` distinction (infra failure halts, content failure falls back). Live host URLs come from `.env` env vars (no new YAML).

**Tech Stack:** Python 3.11, `httpx`, FastAPI + uvicorn, `open_clip` + `torch` (box-only), pytest + `httpx.MockTransport` + FastAPI `TestClient`, Typer CLI, vast.ai RTX 4090.

```python
import math
import httpx


class RemoteAestheticScorer:
    """LAION-Aesthetics v2 scorer. Calls a FastAPI score_server running on the
    vast.ai box (see scripts/score_server/server.py).

    Lifecycle: instantiated once per Stage.run; closed via aclose() at end of stage.
    Mirrors HttpComfyClient -- transport seam for httpx.MockTransport in unit tests.
    """

    def __init__(
        self,
        *,
        host: str,
        timeout: float = 30.0,
        transport: httpx.AsyncBaseTransport | None = None,
    ) -> None:
        if not host:
            raise ValueError(
                "RemoteAestheticScorer requires a host "
                "(set PLATINUM_AESTHETICS_HOST in .env or aesthetics.host in settings.yaml)"
            )
        self._host = host.rstrip("/")
        self._client = httpx.AsyncClient(
            base_url=self._host,
            timeout=timeout,
            transport=transport,
        )

    async def score(self, image_path: Path) -> float:
        with image_path.open("rb") as fh:
            files = {"image": (image_path.name, fh, "image/png")}
            resp = await self._client.post("/score", files=files)
        resp.raise_for_status()
        body = resp.json()
        score = float(body["score"])
        if not math.isfinite(score):
            raise ValueError(f"score_server returned non-finite score: {body!r}")
        return score

    async def aclose(self) -> None:
        await self._client.aclose()
```

```python
@app.post("/score")
async def score(
    image: UploadFile = File(...),
    scorer: Scorer = Depends(get_scorer),
) -> dict[str, float]:
    raw = await image.read()
    if not raw:
        raise HTTPException(400, "empty image upload")
    try:
        # Validate the bytes are a real image.
        Image.open(io.BytesIO(raw)).convert("RGB")
    except (UnidentifiedImageError, OSError) as exc:
        raise HTTPException(400, f"could not decode image: {exc}") from exc
    score_value = float(scorer(raw))
    if not math.isfinite(score_value):
        raise HTTPException(500, "model returned non-finite score")
    return {"score": score_value}
```

```python
"""LAION-Aesthetics v2 model singletons -- runs only on the vast.ai box.

Imports torch + open_clip; do NOT import this module in local CI.
server.py uses lazy `from score_server.model import ...` inside the
`get_scorer` dependency so that local pytest can override the dep
without ever touching torch.
"""

from __future__ import annotations

import io
import os
from pathlib import Path

import torch  # type: ignore[import-not-found]
import torch.nn as nn  # type: ignore[import-not-found]
import open_clip  # type: ignore[import-not-found]
from PIL import Image

DEVICE = "cuda" if torch.cuda.is_available() else "cpu"
MLP_WEIGHTS = Path(
    os.environ.get(
        "LAION_MLP_PATH",
        "/workspace/models/aesthetic/sac+logos+ava1-l14-linearMSE.pth",
    )
)


class AestheticMLP(nn.Module):
    """4-layer MLP head for LAION-Aesthetics v2 (ViT-L/14 features -> score)."""

    def __init__(self, input_size: int = 768) -> None:
        super().__init__()
        self.layers = nn.Sequential(
            nn.Linear(input_size, 1024),
            nn.Dropout(0.2),
            nn.Linear(1024, 128),
            nn.Dropout(0.2),
            nn.Linear(128, 64),
            nn.Dropout(0.1),
            nn.Linear(64, 16),
            nn.Linear(16, 1),
        )

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        return self.layers(x)


# --- Module-level singletons; loaded once on first import --------------------
print(f"[score_server.model] loading CLIP ViT-L-14 on {DEVICE}...")
clip_model, _, preprocess = open_clip.create_model_and_transforms(
    "ViT-L-14", pretrained="openai", device=DEVICE
)
clip_model.eval()

print(f"[score_server.model] loading MLP weights from {MLP_WEIGHTS}...")
_mlp = AestheticMLP(input_size=768)
_mlp.load_state_dict(torch.load(MLP_WEIGHTS, map_location=DEVICE))
_mlp.to(DEVICE).eval()
print("[score_server.model] ready.")


def score_image_bytes(image_bytes: bytes) -> float:
    """Score a raw image (bytes) on the LAION-Aesthetics v2 scale."""
    img = Image.open(io.BytesIO(image_bytes)).convert("RGB")
    with torch.no_grad():
        tensor = preprocess(img).unsqueeze(0).to(DEVICE)
        feats = clip_model.encode_image(tensor)
        feats = feats / feats.norm(dim=-1, keepdim=True)
        return float(_mlp(feats).cpu().item())
```

```python
@app.command()
def keyframes(
    story: str = typer.Argument(..., help="Story id (must have visual_prompts COMPLETE)."),
    scenes: str | None = typer.Option(
        None, "--scenes",
        help="Comma-sep scene indices to run (smoke subset). Default: all.",
    ),
    dry_run: bool = typer.Option(
        False, "--dry-run",
        help="Print the planned scene set + comfy/aesthetics hosts; do not call.",
    ),
) -> None:
    """Generate keyframes for a curator-approved + adapted story.

    Requires visual_prompts COMPLETE. Reads PLATINUM_COMFYUI_HOST and
    PLATINUM_AESTHETICS_HOST from .env (or settings.yaml fallback).
    Use --scenes for smoke runs.
    """
    import logging
    from platinum.models.story import StageStatus, Story
    from platinum.pipeline.context import PipelineContext
    from platinum.pipeline.keyframe_generator import KeyframeGeneratorStage
    from platinum.pipeline.orchestrator import Orchestrator

    cfg = Config()
    story_path = cfg.stories_dir / story / "story.json"
    if not story_path.exists():
        console.print(f"[red]Story not found:[/red] {story} (looked in {story_path})")
        raise typer.Exit(code=1)

    s = Story.load(story_path)
    vp = s.latest_stage_run("visual_prompts")
    if vp is None or vp.status != StageStatus.COMPLETE:
        console.print(
            f"[red]Story {story} has no completed visual_prompts; "
            f"run 'platinum adapt --story {story}' first.[/red]"
        )
        raise typer.Exit(code=1)

    # Parse --scenes
    scene_filter: set[int] | None = None
    if scenes is not None:
        try:
            scene_filter = {int(x.strip()) for x in scenes.split(",") if x.strip()}
        except ValueError as exc:
            raise typer.BadParameter(
                f"--scenes must be comma-separated integers (got: {scenes!r})",
                param_hint="--scenes",
            ) from exc
        max_idx = max(scene_filter, default=-1)
        if max_idx >= len(s.scenes):
            raise typer.BadParameter(
                f"--scenes references index {max_idx} but story has only {len(s.scenes)} scenes",
                param_hint="--scenes",
            )

    comfy_host = cfg.settings.get("comfyui", {}).get("host", "")
    aest_host = cfg.settings.get("aesthetics", {}).get("host", "")

    if dry_run:
        planned = sorted(scene_filter) if scene_filter else list(range(len(s.scenes)))
        console.print(
            f"[cyan]Would generate keyframes for scenes {planned} of story {story}[/cyan]"
        )
        console.print(f"  comfy   = {comfy_host}")
        console.print(f"  scorer  = {aest_host}")
        raise typer.Exit(code=0)

    cfg.settings.setdefault("runtime", {})["scene_filter"] = scene_filter
    ctx = PipelineContext(config=cfg, logger=logging.getLogger("platinum.keyframes"))
    orchestrator = Orchestrator(stages=[KeyframeGeneratorStage()])

    try:
        asyncio.run(orchestrator.run(s, ctx))
    except Exception as exc:
        console.print(f"[red]keyframes failed for {story}: {exc}[/red]")
        raise

    console.print(f"[green]Keyframes complete for {story}.[/green]")
```

## Notes for the executor

1. **TDD discipline.** Every Task 1-22 ends with green pytest. Do not skip the "run-see-fail" step — if the test passes before the implementation, the test isn't really testing the new behaviour.

2. **No `data/stories/*` commits.** Per project memory: regeneratable workspace data must NEVER be committed. Phase 2 produces ~3MB of PNGs locally — leave them on disk, do not stage them.

3. **No `.env` commits.** `.env` is gitignored; verify before each commit.

4. **Subagent scope drift.** If a task feels like it needs to also "clean up" a related thing, RESIST. Fix only what's in the task; surface the cleanup as a follow-up at end of session. Past chore-sweeps (S4 Task 24) accidentally committed workspace data and falsely claimed clean linting.

5. **score_server tests skip torch.** If you find yourself wanting to `pip install torch` to make the tests work, stop — the dependency injection pattern (Task 13) deliberately avoids this. Fix the test pattern, not the requirements.

6. **Live smoke is real money.** Tasks 24-30 spend $1-2 of vast.ai credit. Tear down (Task 30) is mandatory; an instance left running burns ~$10/day.

7. **Reference skills.** During execution, invoke `superpowers:test-driven-development` if any Task feels like it's drifting from RED→GREEN→REFACTOR. Invoke `superpowers:systematic-debugging` if a test stays red after what should be a passing implementation.
